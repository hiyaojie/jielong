## 程序功能
实现输入一个城市名（如深圳），会随机输出另外一个城市名，以前一城市最后一个字开头（能兼容谐音，例如深圳可以接镇江）；
## 运行环境
 0. python3
 1. mongodb
## 使用技术
0. python爬虫
1. python读写excel
2. python操作mongodb
